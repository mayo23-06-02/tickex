# Event Creation Wizard - User Guide

## Overview
The streamlined event creation wizard guides organizers through 4 essential steps to create and publish events on TickEx.

## Step-by-Step Guide

### Step 1: Event Basics
**What you'll enter:**
- **Event Name**: The title of your event (e.g., "Neon Dreams Festival 2025")
- **Organizer**: Your organization or company name
- **Category**: Select from Music Festival, Conference, Workshop, or Sports
- **Start Date & Time**: When your event begins
- **End Date & Time**: When your event ends
- **Venue**: The location name and address

**Tips:**
- Choose a clear, memorable event name
- Ensure dates are in the future
- End date must be after start date

---

### Step 2: Content & Branding
**What you'll enter:**
- **Event Description**: Detailed information about your event
- **Event Banner**: Upload a high-quality image (recommended: 1200×600px)
- **Additional Details**: Any extra information attendees should know

**Tips:**
- Write a compelling description that highlights key attractions
- Use a professional, eye-catching banner image
- Include important details like dress code, parking, etc.

---

### Step 3: Ticket Setup ⭐ (Enhanced)
**What you'll configure:**

For each ticket type:
- **Ticket Name**: e.g., "VIP Pass", "Early Bird", "General Admission"
- **Description**: What's included with this ticket (optional but recommended)
- **Price**: Cost in SZL (use 0 for free tickets)
- **Quantity Available**: Total number of tickets to sell
- **Custom Ticket Design**: Upload a personalized ticket design (optional)

**Ticket Design Specifications:**
- **Dimensions**: 150mm × 80mm (1500px × 800px at 254 DPI)
- **Formats**: JPG, PNG, or WebP
- **Important**: Leave approximately 20% of the right side blank for barcode generation
- **Upload Methods**: 
  - Drag and drop your design file
  - Click to browse and select

**Design Tips:**
✓ Use high-resolution images (at least 1500×800 pixels)
✓ Include your event branding, logo, and colors
✓ Add key event information (date, venue, ticket type)
✓ Ensure text is readable at small sizes
✓ Leave space on the right for the barcode (will be added automatically)

**Example Ticket Layout:**
```
┌─────────────────────────────────────────────────┐
│  [Event Logo]                                   │
│  EVENT NAME                        [BARCODE]    │
│  Date: Dec 25, 2025                [SPACE]      │
│  Venue: City Stadium               [RESERVED]   │
│  Ticket Type: VIP Pass                          │
└─────────────────────────────────────────────────┘
    ← 80% content area →    ← 20% barcode →
```

**Multiple Ticket Types:**
- Click "Add Another Ticket Type" to create different tiers
- Each ticket type can have its own design
- Common examples:
  - Early Bird (discounted, limited quantity)
  - General Admission (standard pricing)
  - VIP (premium pricing, exclusive perks)
  - Student (discounted with verification)

---

### Step 4: Review & Publish
**What you'll see:**
- Complete preview of your event
- Event banner display
- All event details (name, date, location, description)
- List of all ticket types with:
  - Names and descriptions
  - Prices and quantities
  - Custom design previews (if uploaded)

**Final Actions:**
- Review all information carefully
- Click "Confirm & Publish Event" to go live
- Your event will immediately appear on the landing page
- Tickets will be available for purchase

**After Publishing:**
- Event appears on your dashboard
- Customers can discover and purchase tickets
- You can track sales and metrics in real-time

---

## Navigation

### Moving Between Steps
- **Continue**: Click to move to the next step
- **Back**: Click to return to the previous step
- **Progress Bar**: Shows your current position in the wizard

### Saving Your Work
- All data is temporarily stored as you navigate between steps
- Data is only saved to the database when you click "Confirm & Publish"
- If you leave the wizard, your progress will be lost

## Common Questions

**Q: Can I edit my event after publishing?**
A: Currently, events cannot be edited after publishing. This feature is coming soon.

**Q: Do I need to upload a custom ticket design?**
A: No, it's optional. If you don't upload a design, a default template will be used.

**Q: What happens if I don't have a ticket design ready?**
A: You can publish without a custom design and add it later (feature coming soon).

**Q: How many ticket types can I create?**
A: There's no limit! Create as many as you need for your event.

**Q: Can I offer free tickets?**
A: Yes! Set the price to 0 for free tickets.

**Q: What image formats are supported for ticket designs?**
A: JPG, PNG, and WebP formats are all supported.

**Q: How do I ensure my ticket design looks good?**
A: Use the preview feature in Step 3 to see how your design will appear. Make sure text is readable and the layout is clean.

## Troubleshooting

**Issue: Date picker not working**
- Ensure you're clicking inside the date input field
- Try typing the date manually in the format shown

**Issue: Image upload failing**
- Check that your file is under 10MB
- Ensure the file is a valid image format (JPG, PNG, WebP)
- Try a different browser if the issue persists

**Issue: Can't proceed to next step**
- Ensure all required fields are filled
- Check that dates are valid (start before end)
- Verify at least one ticket type is configured

## Best Practices

✓ Keep it concise but descriptive
✓ Include the year if it's an annual event
✓ Avoid special characters that might cause issues

### Pricing Strategy
✓ Offer early bird discounts to drive early sales
✓ Create tiered pricing (General, VIP, Premium)
✓ Consider student or group discounts

### Ticket Design
✓ Match your event branding
✓ Use high-contrast colors for readability
✓ Include essential information only
✓ Test the design at actual size before uploading

✓ Include practical information (parking, accessibility)
